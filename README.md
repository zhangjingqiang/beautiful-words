# Installation

```
$ mkvirtualenv beautiful_words
$ pip install -r requirements.txt
$ python manage.py sqlmigrate words 0001
$ python manage.py migrate
$ python manage.py createsuperuser
$ python manage.py runserver 0.0.0.0:8080
```
